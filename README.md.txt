Catálogo de Juegos

Este proyecto es una aplicación web desarrollada en Angular cuyo objetivo es presentar un catalogo interactivo de juegos.
Permite al usuario explorar una lista de juegos, ver detalles al hacer clic y navegar entre distintas secciones de la app utilizando rutas.
